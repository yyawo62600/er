十七种心理品类如下：

羞愧、内疚、冷漠、悲伤、恐惧、欲望、愤怒、骄傲，这些叫做负面心理，又叫负能量；还有勇气、淡定、主动、宽容、明智、爱、宁静、平和、觉悟，这些叫做正面心理，又叫正能量。
